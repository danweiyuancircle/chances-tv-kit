# @chancestv/tv-ui

面向低版本 WebView（Chromium 53+）的 Vue 3 TV 端基础组件库（全部 E 前缀），基于 [`@chancestv/tv-focus`](https://www.npmjs.com/package/@chancestv/tv-focus) 的空间导航焦点系统。业务层只用这一层即可。

## 文档

完整文档、安装、用法、组件清单与在线 Demo 见 GitHub 仓库（npm 页不再维护文档，以仓库为准）：

- 仓库与文档：https://github.com/danweiyuancircle/chances-tv-kit
- 在线 Demo（遥控器方向键操作）：https://danweiyuancircle.github.io/chances-tv-kit/
